10 ways to make your HTML-CSS code accessible

Web accessibilty is the most talked topic these days. But do you know why accessibilty is important. One of the main reasons is that because of accessibility people with disabilities are able to use the same platforms and features that everyone else can. The whole web is beneficial fully only if its universally accessible to all. Some simple steps to make your webpage accessible are listed below:

  • Use heading tags to show hierarchy in your content : This is an important way to accessible content. Make sure that main heading comes in h1 tag and then its subsections can be h2, h3 and so on.
  • Add a 'main' tag for inputting the most specific content of the page. ALso, there is only one main tag per page.
  • Add alternative text to all images on the webpage: This mainly helps incase the image is broken or when screen readers are used for reading the page.
  • Wrap your content in 'article' tag when you are writing independant content like news , blogs , etc.
  • Contect inside the article element can be grouped inside the section elements. This ways thematically related contect can be put together as sections.
  • The figure element can be used to wrap visual content like images,diagrams or charts in a semantic way.
  • A particular input element can be put inside its respective label element to provide it a specific form control.
  • Links can be made more navigable by adding the accesskey attribute within the links, button, form control elements.
  • Use various color combinations and contrasts to provide better readability.
  • Use header and footer elements to provide navigation links and copyright information respectively.

In all, all these steps provide a semantic meaning to our page hence increasing its accessibilty.
